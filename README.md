# robot-trainer

Student-facing delivery site for Mercersburg Academy Robotics — course and club.

This repo holds the site, routing, auth, per-student progress, tracking, and page
assembly. It holds **no curriculum**. Lessons, designs, Learning Mode prompts, and
book edits live in `Weymuth/zumo`. robot-trainer reads from content repos and
reports problems; it never writes to them.

Quizzes, grades, and static handouts stay in Canvas.

Project state: see `LIVE_ROBOT_TRAINER.md`. That file wins over anything else,
including this one.
